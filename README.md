# lofi-react

Vite + React + TypeScriptで作成したローファイ音楽販売アプリ（開発中）

## 概要

ローファイ音源を一覧表示し、カートに追加できるWebアプリです。
今後、音源再生・決済機能を実装予定です。

## 機能

- 曲一覧表示
- カート追加機能
- 商品カードUI

## 技術スタック

- React
- TypeScript
- Vite
- GitHub / Vercel

## 今後の予定

- 音源再生機能
- Supabase連携
- 決済機能（Stripe or Shopify）
